## 🛠️ Projects

### Hyprx 🕹️
[![Hyprx](https://img.shields.io/badge/Hyprx-Python-blue?style=for-the-badge)](https://github.com/zPleum/Hyprx)  
A free & improved **Minecraft CLI tool** inspired by Banana.  
- Crawls servers & proxies  
- Monitors players & bot commands  
- Fully GPLv3 open-source

### Hyprv ⚡
[![Hyprv](https://img.shields.io/badge/Hyprv-Java-blue?style=for-the-badge)](https://github.com/zPleum/Hyprv)  
Velocity proxy plugin for **packet hooking & logging**.

- Uses Java `MethodHandles` & Velocity internal APIs  
- GPLv3 licensed
